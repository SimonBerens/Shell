# CShell
#### CrystalSimonhell
Making a shell in C

## How to Run
1. In a bash shell, run:
  `$ make`
  `$ make run`
  
## Working Features
* cd
  * Needs a '/' at the end of the path to work
* exit
* Basic bash commands
  * e.g. ls, ps, etc.
* Semicolon parsing
  * Supports multiple commands
  * No spaces after each semicolon
* Shows current working directory
* Supports arguments for commands
